2018 - 2024 The BMJ. AWS Cloud Engineer/DevOps. The British Medical Journal is one of the most prestigious publications in its field. The BMJ publishes 60 journals in paper and digital form and a clinical decision support tool called "Best Practice" which is the product I have been most closely associated with.

Initially the app was unreliable and would alert the operator on call most nights, via Pager Duty. Sometimes that would be me. I installed an ELK stack, and traced the issue to a combination of EC2 instances having insufficient memory, Xmx being set incorrectly for the Java processes and in one case a Java library which had a memory leak.

Having achieved stability I could then right size the EC2 instances and move to ARM architecture to save money and carbon emissions. Subsequently, compute loads were migrated to Kubernetes using EKS/ECS.
